## <font color='#1E72AD'>NSArray+MFExtras</font>

***


### -arrayByRemovingLastObject
<br>
Returns the array without its last object.<br>
Returns an empty array if array contains only one object.
<br><br>
<tt>**-(NSArray*)arrayByRemovingLastObject;**</tt>
<br><br>

-------


### -arrayByRemovingFirstObject
<br>
Returns the array without its first object.<br>
Returns an empty array if array contains only one object.
<br><br>
<tt>**-(NSArray*)arrayByRemovingFirstObject;**</tt>
<br><br>

-------

### -alphabeticallySortedArray

<br>
Returns strings in the array sorted alphabetically.<br>
<br>
<tt>**-(NSArray*)alphabeticallySortedArray;**</tt>
<br><br>
<font color='#FF3221'>Array must contains only NSString objects.</font>
<br><br>

-------

### -reversedArray

<br>
Returns a new array with the array elements in reverse order.<br>
<br>
<tt>**-(NSArray*)reversedArray;**</tt>
<br>
